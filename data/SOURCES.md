# Sources for rulers & tenure dates

Working bibliography for `countries/*.json` leader sequences (Soviet-era republic heads and post-independence heads of state).  
**Not** a claim of primary-source verification — entries were compiled mainly from English Wikipedia office lists and cross-checked with [rulers.org — Soviet republics](http://rulers.org/sovrep.html) where useful.

For act-level citations keyed by id (sovereignty / independence documents), see [`sources.json`](./sources.json). Link those ids from country files via `"sources": ["id"]`.

Accessed: **2026-07** (month–day enrichment and succession-gap pass).

## How to read this

| Tier | Meaning |
|------|---------|
| **Primary list** | Main page used for presidential / head-of-state succession |
| **Soviet-era** | First Secretaries / republic chairs for 1985–independence |
| **Cross-check** | Extra lists used when filling short interim or transitional gaps |

Wikipedia pages change. Prefer inauguration / resignation days from those lists; treat late-Soviet First Secretary **days** as weaker than presidential inaugurations (see [`README.md`](./README.md) methodology).

## Unrest / conflict layer (`unrest` arrays)

Event names and year ranges are secondary-source sketches (Wikipedia conflict articles). Treat the unrest layer as **experimental**; do not cite decade-long war bars as precise campaign chronologies without further checking.

## Updating this file

When you correct a tenure from a new page, add that URL under the country and a one-line note in [`CHANGELOG.md`](./CHANGELOG.md). Prefer keeping [`sources.json`](./sources.json) for citeable act documents, and this file for succession lists.
